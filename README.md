## 📌 About The Project

This is the **admin panel** of the e-commerce application.
It allows administrators to manage products, orders, and overall platform operations efficiently.

---

## ✨ Features

* Admin login / logout
* Add new products
* Update product details
* List products
* Delete products
* View and manage orders
* Monitor user activities
